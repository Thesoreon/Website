Co dokážu vytvořit pomocí CSS?
Introduction
CSS definuje, jak by se měli HTML elementy zobrazovat
Syntaxe: Selector {Property: value;}
Comments
/* Comment */
Inserting CSS
External, Internal nebo inline
Colors
Predefinovana jmena nebo A/RGB, HEX, HSL/A hodnoty
Backgrounds
Properties:
background-color: Nastaveni barvy
background-image: nastaveni obrazku jako pozadi
background-repeat: opakovani pozadi(repeat-x/y!)
background-attachment: Nastavi, zda ma obrazek scrollovat(fixed)
Borders
Border-style: typ borderu: dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden
border-radius: kulaté rohy
Margins
Určují místo mezi elementy
Paddings
Určují místo mezi kontentem
Height and Width
Max-height, max-width
Box Model
Kontent - Padding - Border - Margin
Outline
outline-style: typ: otted, dashed, solid, double, groove, ridge, inset, outset, none, hidden
Text
Text-align
Text-decoration
Text-transform: Velká, malá písmena
Text-indentation: posunutí prvního řádku
text-spacing: misto mezi pismeny
line-height: mezery mezi radky
Text-shadow: horizontalni, vertikalni, barva
Fonts
Font-family: "Times new Roman" atd.
Font-style: normal, italic
Font-weight: bold, normal
Links
a:visited - Link, ktery byl uz rozkliknut
a:active - Link, ktery je zrovna nakliknut
Display
Block elementy využívají maximální šířky, co můžou
Inline elementy využívají šířky, kterou potřebují
Position
Static - Neni ovlivněno top, left, right, bottom properties
Relative - Je polohováno vzhledem k jeho původní pozici
Fixed - Element je lokalizovan podle viewportu
Absolute - Je lokalizovan relative podle nejblizšího předka
Sticky - Chová se jako fixed, s tím, že se "přilepí" když k němu dojde scrollbar
Elementy se mohou překrývat, proto se využívá z-index
Overflow
Overflow: hidden, visible, scroll, auto
Float
float: inherit, right, left
clear
Align
Div elementy se zarovnavaji pomoci: margin: auto
Combinators
Child selector: napriklad: "div > p"
Adjacent Sibling selector: napriklad: div + p = První paragraph po divu je bran jako součást divu
General Siblings Sector: napriklad: div ~ p = Všechny paragrafy, ktere jsou "sourozenci" divu
