Co dokážu vytvořit pomocí CSS?
NAZEV CO TO UMI SYNTAXE
Introduction CSS definuje, jak by se měli HTML elementy zobrazovat Syntaxe: Selector {property: value}
Inserting CSS Tři typy: external, internal, inline Attributy: style Tagy: <link>, <style>
Colors Barvy lze zadat různými způsoby. RGB, ARGB, HSL, HSLA, Hexadecimálně, předdefinovanými jmény Příklady: #FFFFFF, RGB: 256, 256, 256
Backgrounds Pozadí lze nastavit většině elementů Properties: background-color: nastavení barvy, background-image: nastaveni obrazku jako pozadi, background-repeat: opakovani pozadi(repeat-x/y!), background-attachment: Nastavi, zda ma obrazek scrollovat(fixed)
Borders Nastaví ohraničení elementu Border-style: typ borderu: dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden Border-radius: kulaté rohy
Margins Nastaví vnější místo mezi elementy Margin: top, right, bottom, left
Paddings Nastaví místo mezi kontentem elementu a okolím Padding: top, right, bottom, left
Height and Width
Box model Grafické znázornění, jak jsou elementy zobrazované Kontent - Padding - Border - Margin
Outline Zobrazení čáry okolo borderu outline-style: typ: otted, dashed, solid, double, groove, ridge, inset, outset, none, hidden
Text Úprava a vzhled textu Text-align Text-decoration Text-transform: Velká, malá písmena Text-indentation: posunutí prvního řádku text-spacing: misto mezi pismeny line-height: mezery mezi radky Text-shadow: horizontalni, vertikalni, barva
Fonts Nastavení fontu pro text Font-family: "Times new Roman" atd. Font-style: normal, italic Font-weight: bold, normal
Links Nastavení elementu s odkazem a:visited - Link, ktery byl uz rozkliknut. a:active - Link, ktery je zrovna nakliknut
Display Jak by se měli elementy zobrazovat block/inline Block elementy zabírají největší šířku, co můžou. Inline elementy mají šířku, kterou potřebují
Position Nastavení pozice elementu Static - Neni ovlivněno top, left, right, bottom properties. Relative - Je polohováno vzhledem k jeho původní pozici. Fixed - Element je lokalizovan podle viewportu. Absolute - Je lokalizovan relative podle nejblizšího předka. Sticky - Chová se jako fixed, s tím, že se "přilepí" když k němu dojde scrollbar. Elementy se mohou překrývat, proto se využívá z-index
Overflow Jak by měli elementy přetékat Overflow: hidden, visible, scroll, auto
Float Nastaví obtékání elementu float: inherit, right, left
Align Zarovnání elementu Div elementy se zarovnavaji pomoci: margin: auto
Combinators Slouží k získání selectoru "rodičů" atd. Child selector: napriklad: "div > p". Adjacent Sibling selector: napriklad: div + p = První paragraph po divu je bran jako součást divu. General Siblings Sector: napriklad: div ~ p = Všechny paragrafy, ktere jsou "sourozenci" divu
