Co dokážu vytvořit pomocí CSS?
NAZEV CO TO UMI SYNTAXE
Introduction CSS definuje, jak by se měli HTML elementy zobrazovat Syntaxe: Selector {property: value}
Inserting CSS Tři typy: external, internal, inline Attributy: style Tagy: <link>, <style>
Colors Barvy lze zadat: RGB, ARGB, HSL, HSLA, Hexadecimálně Příklady: #FFFFFF, RGB: 256, 256, 256
Backgrounds Pozadí lze nastavit většině elementů Properties: background-color, image, repeat, attachment
Borders Nastaví ohraničení elementu Border-style, radius
Margins Nastaví vnější místo mezi elementy Margin: top, right, bottom, left
Paddings Nastaví místo mezi kontentem elementu a okolím Padding: top, right, bottom, left
Height and Width
Box model Grafické znázornění, jak jsou elementy zobrazované Kontent - Padding - Border - Margin
Outline Zobrazení čáry okolo borderu outline-style
Text Úprava a vzhled textu Text-align, decoration, transform, indentation, spacing
Fonts Nastavení fontu pro text Font-family, Font-style
Links Nastavení elementu s odkazem a:visited, a:active
Display Jak by se měli elementy zobrazovat block Block elementy zabírají největší šířku, co můžou.
Position Nastavení pozice elementu Static, relative, absolute, fixed, sticky
Overflow Jak by měli elementy přetékat Overflow: hidden, visible, scroll, auto
Float Nastaví obtékání elementu float: inherit, right, left
Align Zarovnání elementu Div elementy se zarovnavaji pomoci: margin: auto
Combinators Slouží k získání selectoru "rodičů" atd. Child selector: div > p, Adjacent Sibling: div + p
Pseudo Class Využití při speciálních eventech link, visited, hover, active, first-child, lang
Pseudo elements Slouží k stylování první věty v divu first-line, first-letter, before, after, selection
Opacity Úprava transparentnosti elementů, hodnota od 0 do 1 opacity: value;
DropDown Zobrazí další kontent při přejetí myší nad aktivátorem Využití property display
Tooltip Zobrazí informace o nečem, když přes to uživatel přejede myší Využití property display
Attribute Selectors Slouží k stylování specifického attributu určeného selectoru Syntaxe: selector[attribute]
