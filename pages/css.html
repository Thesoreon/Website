Rounded, image-borders
Rounded borders = Property: border-radius: value;
Image borders = Property: border-image: url(example.url) int round;
Backgrounds
Multiple backgrounds = Property: background-image: url(example.png), url(example2.png);
Size of backgrounds = Property: background-size: height, width; Další hodnoty: contain a cover.
Origin of backgrounds = Property: border-box, padding-box, content-box;
Background clip = Property: border-box, padding-box, content-box;
"Full Size background image" = využití html selectoru a hodnoty cover.
