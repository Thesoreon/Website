CSS3

Introduction
CSS3 je rozděleno do tzv. "Modulů"
  • Selectors
  • Box Model
  • Backgrounds and borders
  • Image Values and replaced content
  • Text effects
  • 2D/3D transformations
  • Animations
  • Multiple Column Layout
  • User interface
Rounded, image-borders
Rounded borders = Property: border-radius: value;
Image borders = Property: border-image: url(example.url) int round;
Backgrounds
Multiple backgrounds = Property: background-image: url(example.png), url(example2.png);
Size of backgrounds = Property: background-size: height, width; Další hodnoty: contain a cover.
Origin of backgrounds = Property: border-box, padding-box, content-box;
Background clip = Property: border-box, padding-box, content-box;
"Full Size background image" = využití html selectoru a hodnoty cover.
Gradients
Gradients nám dovolují vytvořit přechody mezi dvěma a více barvami
Linear - jdoucí od down/up/left/right/diagonally
Radial - definovány jejich středem
Syntax linear = Property: background: linear-gradient(směr, barva, barva,..);
Syntax radial = Property: radial-gradient(shape size, barva, barva)
Shadows
Text shadow = Property: text-shadow: horizontalShadow verticalShadow blur color;
Box shadow = Property: box-shadow: horizontalShadow verticalShadow blur color;
Pro více stínů na jednom elementu se dělí čárkou
Text
Text overflow = Property: text-overflow: ellipsis/clip;
Word wrap = Property: word-wrap: break-word;
Word break = Property: break-all, keep-all;
Fonts
Font formáty: TTF, OTF, WOFF, SVG, EOT
@font-face rule, Property: @font-face {font-family: name; src: url(example.woff); }
Transforms 2D and 3D
  • Translate
  • Rotate
  • Scale
  • SkewX
  • SkewY
  • Matrix
Translate(x, y);
Rotate(deg);
Scale(hodnota, hodnota);
Skew(deg, deg);
Matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())

For 3D Transform:
RotateX, RotateY and RotateZ
Transitions
Transitions dovolují průběžné měnění properties z jedné hodnoty na druhou při daném čase

Speed Curve of the Transition
  • Ease
  • Linear
  • ease-in
  • ease-out
  • ease-in-out
  • cubic-bezier(n, n, n, n)
Shorthand-property: Transition delay, duration, property, timing function;
Animations
Animations dovolují průběžné měnení properties z jedné hodnoty na druhou (několik na jednou, kolikrát chceme)
Pro použití animací musíme prvně specifikovat keyframes pro animaci.
Syntax = @keyframes nazev {from {property: value; } to {property: value; } }
Pro použití animace, musíme ji nabindovat k elementu = animation-name: nazev; animation-duration: 0s; animation-delay: 0s; animation-iteration-count: infinite; animation-timing-function
Speed Curve of the Animation
  • Ease
  • Linear
  • ease-in
  • ease-out
  • ease-in-out
  • cubic-bezier(n, n, n, n)
Images
Border-radius: value; Rensposive image = Property: selector {max-width: 100%; height: auto;}
Center image = Property: selector {display: block; margin: auto;}
Flipped image = Property: transform: scaleX(-1);
Object Fit
Object-fit definuje, jak se měla změnit velikost obrázku či videa, tak aby vyplnila svůj container
Property: object-fit: cover, contain, fill, none, scale-down;
Multi-column Layout
Property: column-count: value; column-gap: value; column-rule-style; column-rule-width; column-rule-color;
Shorthand property: colum-rule-width column-rule-style column-rule-color;
User interface
Definuje zda může být nějaký element upravován uživatelem.
Resizing = Property: resize: horizontal/vertical/both;

Box sizing
Dovoluje nám připočítat padding a border properties do celkové výšky a šířky elementu.
Box-sizing: border-box;
Flexbox
Nejnovější způsob layoutu v CSS3.
Flex container = Container musí mít nastaveno tyto properties: Display: flex/inline-flex;
Flex items = jejich "rodičem" je flex container, který jim říká, jak mají býti uspořádány.

Properties:
  • row-revers
  • column
  • column-reverse
  • justify-content
  • space-between
  • space-around
  • align-items
  • align-self
  • flex
Media Queries
Použití pro vytvoření responsivního layoutu při různých rozlišeních. Syntax = @Media (podmínka) {code}
