Headings Vytvoří nadpis, užitečný pro search engines <h1> - <h6>
Paragraphs Udělá z textu odstavec <p>

Quotation Vytvoří z textu citát <q>, <blockquote>, <abbr>, <address>, <cite>, <bdo>
Links Odkaz na soubor či hypertextový odkaz Attributy: _blank, _self, _parent, _top, framename, href, src. Tagy: <a>, <img>. Lze vytvořit bookmarky přidáním ID elementu do href attributu
Images Vloží obrázek na webovou stránku, area: obrazek s klikatelnymi oblastmi, picture: "pole" obrázků Attributy: src, alt. Tagy: <img>, <area>, <picture>
Tables Vytvoří tabulkový layout, který jde následně upravit pomocí CSS <table> rows: <tr> header: <th> table cell: <td>, <colspan>, <rowspan>
Lists Vytvoří list (seznam) věcí unordered list: <ul>, ordered list: <ol>, description list: <dl>
Blocks Blockové elementy zabírají vždy největší šířku, co mohou <div>, <h1-6>, <p>, <form>...
Inline Inline elementy zabiraji sirku, co potrebuji <span>, <a>, <img>
